# Binary Classification

This project builds a model to classify data into two classes (0 or 1).

## Dataset
- File: classification.csv
- 250 samples
- 100 features
- Last column is the target (0 or 1)

## What the project does
- Loads and prepares the dataset
- Trains two different classification models
- Tests performance on unseen data
- Chooses the best model based on results

## How to run
1. Install required libraries
2. Open the notebook:
3. Run `classification.ipynb`

## Libraries used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
